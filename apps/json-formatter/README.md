# JSON Formatter

A compact single-editor JSON tool.

## Route

- `.../apps/json-formatter`

## Features

- One editor box for both input and output
- `Format` with indentation options
- `Minify`
- `Validate`
- `Copy`
- `Clear`
- Inline error/status messages

## Usage

1. Paste JSON into the editor.
2. Choose indentation from the top dropdown (default: 2 spaces).
3. Use top action buttons.
4. If JSON is invalid, an inline error appears and content stays unchanged.

## Files

- `app.js`: app UI and behavior
- `styles.css`: app-specific styles
